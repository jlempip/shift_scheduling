Aim of this project is to create a working scheduler system for use in managing shifts of the employees at my company.

Minimum requirements for the program at the moment are:
- Manage scheduling shifts for 4 weeks at a time, atleast 2 weeks before the first shift
- Specify number of people needed for each shift in the range 2-10
- Specify a certain employee cannot work a certain shift
- Specify a miminum and maximum amount of hours during the 4 week timespan for each individual employee
- A certain employee should not be scheduled in the same location for two weeks in a row (2 locations)
- Each shift needs atleast one employee with emergencency first aid training
